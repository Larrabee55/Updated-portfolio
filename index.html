Star Wars Hangman

This assignment was coded the third week of the bootcamp after learning Javascript! If the user gueses wrong it puts it the the letters guessed wrong area. If the user input 13 wrong letters then it prompts the user that they lost and asks with they want to play again. If the user guesses the word before they run out of guesses it adds one to the wins variable then it prompts the user they won and if they want to play again. If the user hits play again it picks another random word from the array for the user to guess.
